**<u>Purpose</u>**

The final project provides students with an opportunity to integrate
theoretical understanding and practical skills in Machine Learning.
Students are required to design, implement, deploy, and evaluate a
machine learning solution that addresses a real-world problem.

This project emphasizes an **end-to-end classical machine learning
pipeline**, including data understanding, model development, system
deployment, and evaluation through **real user interaction**.

**<u>Learning Outcomes (LOs)</u>**

Upon successful completion of this project, students will be able to:

- **LO1**: Explain fundamental machine learning concepts and challenges.

- **LO2**: Analyze datasets and apply regression, classification, or
  clustering techniques appropriately.

- **LO3**: Implement, deploy, and evaluate a machine learning system
  based on real user interaction and feedback.

**<u>Mapping of Deliverables to Learning Outcomes</u>**

| **Deliverable**                      | **Aligned Learning Outcomes** |
|--------------------------------------|-------------------------------|
| Machine Learning Model & Source Code | LO1, LO2                      |
| Deployed Application                 | LO2, LO3                      |
| Report (PPT)                         | LO1, LO2, LO3                 |
| Demo Video                           | LO3                           |

> **Suggested Project Themes**

- Regression-based predictive systems

- Classification-based decision support systems

- Recommender systems with user interaction

- Clustering-based user segmentation tools

- End-to-end machine learning applications with user feedback loop

- 

**<u>Important Note</u>**

Only **classical machine learning methods** are allowed (linear models,
tree-based models, SVM, clustering). **Deep learning approaches are
strictly not allowed** in this course.

**<u>Technical Notes</u>**

- Programming Language: Python

- Libraries: NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn

- Deployment Frameworks (Mandatory): Flask, FastAPI, or Streamlit

- Version Control: Git (mandatory)

**<u>Deployment Requirement (Mandatory)</u>**

- The machine learning model **must be deployed** as a usable
  application.

- Deployment may be web-based or API-based.

- The Latency of the system should be **less than 100 ms**.

- The deployed system must:

  - accept user input,

  - execute the trained model,

  - produce clear and interpretable output.

- Accessibility may be via:

  - local demo,

  - shared access link,

  - or recorded live demonstration.

**<u>Real User Testing Requirement (Mandatory)</u>**

- Minimum **5 real users**, not part of the project team.

- Students must define:

  - anonymized user background,

  - usage scenario,

  - usability and usefulness feedback.

- Collect **quantitative** (rating/Likert) and **qualitative** feedback.

- Results **must be analyzed and presented in the PPT**.
